Producto Escalar entre arreglos

  • Cargue dos arreglos de dimensión N números
  • La dimensión del arreglo es ingresada por el usuario
  • Los elementos (números) del arreglo son ingresados por el usuario
  • Calcule el producto escalar entre los dos arreglos: Almacene en un arreglo de dimensión N números.
  • Si A y B el producto es a1*a2 + b1*b2 + c1*c2
Ejemplo: N = 3 v1 = 0, 1, 2 v2 = 3, 4, 5 producto = 14
